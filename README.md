# Односторінковий сайт «Хитрий Лис»

Готовий статичний сайт бренду без зовнішніх залежностей. Усі зображення та шрифти зберігаються локально.

## Як відкрити

Відкрийте `index.html` у сучасному браузері або запустіть простий локальний сервер у папці сайту:

```bash
python3 -m http.server 8080
```

Після цього сайт буде доступний за адресою `http://localhost:8080`.

## Що вже працює

- адаптивна верстка для комп’ютера й телефона;
- віковий екран 18+;
- мобільне меню;
- плавні анімації появи;
- інтерактивні картки шести напоїв;
- модальні вікна з легендами;
- локальні шрифти й оптимізовані WebP-зображення.

## Перед публікацією

Додайте підтверджені контакти, адресу, посилання на соціальні мережі та юридичні дані. За потреби кнопки можна під’єднати до бронювання, каталогу або форми для партнерів.
